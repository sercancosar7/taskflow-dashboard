# TaskFlow Dashboard

A real-time task management dashboard built with React and TypeScript. Features Kanban boards, analytics, and team collaboration tools.

## Features

- Interactive Kanban board with drag-and-drop
- Real-time analytics and reporting
- Task filtering and search
- Team activity feed
- Responsive design

## Tech Stack

- React 19 + TypeScript
- Tailwind CSS
- Recharts for data visualization
- Vite build system

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
